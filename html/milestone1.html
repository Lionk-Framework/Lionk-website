Milestone 1

Monday, 22th July 2023

The first milestone of our project was reached after two weeks of work, which allowed us to establish an environment conducive to the development of Lionk Core. The basic architecture is defined, and the first proofs of concepts have been completed. We are proud of what we have accomplished. Here are the deliverables of this first milestone:

Deliverables

Preliminary architecture description Mockups (Figma, paper-pencil, etc.) / Landing page Description of technical choices Description of the work process (e.g., git flow, devops, etc.) Implementation of development tools (VCM, Issue tracker) Implementation of a deployment environment Implementation of a delivery and deployment pipeline (CI/CD) Demonstration of the deployment of a modification
